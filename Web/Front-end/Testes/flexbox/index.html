Flex Item

Apartir de agora as propriedades utilizadas modificará os itens, ou seja, os filhos diretos do flex container (pai). Ou seja, as propriedades precisam ser aplicadas em cada filho.

flex-grow: (number);

Define quanto o flex item irá crescer proporcionalmente, com o objetivo dos itens sempre atingirem 100% do pai (container). Sendo 0 seu valor padrão, porém, sempre respeitando o tamanho do conteúdo.
